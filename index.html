Frequently Asked Questions

Is iBrain really free?
Yes. iBrain is completely free to download and use. All local AI features work without payment. The optional cloud API integration (Claude, GPT) requires your own API key, but the local Ollama features are 100% free.
How does offline mode work?
iBrain connects to a local Ollama instance running on your Mac. AI models are downloaded to your machine and processed locally using your CPU/GPU. No internet is required after the initial model download.
Can I use cloud AI instead?
Yes. iBrain supports optional cloud APIs — bring your own Anthropic (Claude) or OpenAI (GPT) key. This is useful for more advanced models, but completely optional. All local features work without it.
Does it work on Intel Macs?
iBrain is optimized for Apple Silicon (M1, M2, M3, M4) and requires macOS 14 (Sonoma) or later. Intel Mac support is not available due to local AI performance requirements.
Is iBrain part of iSuite Office?
Yes. iBrain is included free with the iSuite Office bundle. If you already own iSuite, you get iBrain at no extra cost. It also integrates with iVoz for voice-to-AI workflows.
